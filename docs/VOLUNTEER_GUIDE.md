# Volunteer guide — editing The River Church website

This is for anyone on the church team who wants to update the website (typo fixes, new announcements, photo swaps, schedule changes). You do **not** need to know how to code.

---

## Where to log in

Go to **`https://theriverag.church/admin/`**.

You'll be asked to sign in with GitHub. The first time you do this:

1. If you don't already have a GitHub account, you'll need to create one (free, takes 60 seconds).
2. Send your GitHub username to Jordan or Steve so we can give you access.
3. Once added, the admin login will work and remember you next time.

---

## What you can edit

After logging in, you'll see a sidebar with these sections:

### 🏠 Site Settings
The text that shows on every page. Edit these to update the site-wide content:
- **Announcement Bar Text** — the gold bar at the very top of every page
- **Hero Tagline** — the paragraph below "YOU BELONG HERE!" on the home page
- **Footer Tagline** — the short sentence under the logo at the bottom
- **Phone Number / Email** — contact info shown in the footer
- **Address** — shown in the footer
- **Facebook / Instagram / YouTube URLs** — the social icons in the header

### 🕐 Service Times
The three cards on the home page. Edit if a service time changes or you add a new one. Each card has:
- **Card Heading** — e.g. "Sunday Services 9:30 & 11am"
- **Time / When** — e.g. "Sundays @ 9:30 & 11:00 AM"
- **Description** — the paragraph under the time

### 👤 About — Leadership
Pastors Steve and Tracey's bio on the About page. You can:
- Change the section heading
- Replace the photo (drag and drop a new one)
- Edit any of the bio paragraphs

### 📝 Announcements & Blog
Add new announcements or blog posts. Click **"+ New Announcements & Blog"** to create one. Each post has:
- **Title** — the headline
- **Publish Date**
- **Featured Image** — optional
- **Summary** — 1–2 sentences shown in the list
- **Body** — the full post (with formatting, headings, links, images)
- **Author** — optional

### 📷 Photo Galleries
Swap the home page photos:
- **Hero Image** — the big background photo behind "YOU BELONG HERE!"
- **Gallery Strip 1** — the top row of 3 photos
- **Gallery Strip 2** — the bottom row of 3 photos (below "Are You Curious About Jesus?")

---

## How to make a change

1. Open the section you want to edit (e.g. **Site Settings**).
2. Make your changes in the form.
3. Click **Save** — your edit becomes a draft.
4. When you're ready, click **Publish** at the top right.
5. The website will update automatically within about a minute.

---

## "Editorial Workflow" — drafts and review

Saves go through three stages:

1. **Draft** — you saved but haven't asked for review yet
2. **In Review** — Jordan or Steve sees the change in the admin panel and can approve or send it back
3. **Ready** — approved, will publish to the live site

This means you can work on a draft, set it aside, and come back to it later. You can also send a draft to someone else for a second opinion before it goes live.

## What you should NOT edit (yet)

| Thing | Why |
|---|---|
| Sermon list | Updates automatically from YouTube every day. Don't edit by hand. |
| Calendar / events | Lives in Planning Center. Edit there, the website auto-syncs. |
| Beliefs page | Keep verbatim with AG.org. If you need to update wording, ask Steve. |
| Anything in the **"sermons"** folder | Auto-generated from the YouTube channel. |

---

## If something looks broken

- The website won't go down from anything you do in the admin panel — every save creates a draft, not a live edit.
- If a published change looks wrong, log in, go to the section, and revert. Or send a screenshot to Jordan and we'll roll it back from the git history.
- Worst case: every change is logged forever and can be undone in seconds.
