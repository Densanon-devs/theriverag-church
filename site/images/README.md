## How to fill the gaps

1. **Logo PNG/SVG** — open `Logo/River Logo Brand.ai` in Illustrator → Export As → PNG (transparent, 600×600 or larger). Save as `logo.png`.
2. **Favicon** — same logo mark, exported at 256×256 PNG, then converted to .ico.
3. **Photos** — ask Caitlyn / Steve for the church's photo library, OR pull screenshots from recent YouTube livestreams and Facebook/Instagram posts:
   - Facebook: https://www.facebook.com/theriverag
   - Instagram: https://www.instagram.com/theriverag.church
   - YouTube: https://www.youtube.com/@theriverag.church/streams
4. **Currently live Squarespace site** — gallery images are still hosted at squarespace-cdn.com URLs. You can right-click → Save image as on the live site at https://theriverag.church before the cutover.
